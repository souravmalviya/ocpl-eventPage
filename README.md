# OCPL 2026 Timeline

This is a React + Vite project for the OCPL 2026 roadmap UI.

## Requirements

- Node.js 18+
- npm 9+

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in terminal (usually http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

The production output is generated in the `dist/` folder.
